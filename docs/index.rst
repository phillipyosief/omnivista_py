Welcome to the omnivista-py documentation!
===================================

**omnivista-py** is a Python library designed to simplify interaction with the OmniVista API. 
It provides an easy-to-use interface for authenticating with the API, managing network devices, and querying performance data. This library is ideal for network administrators and developers who need to automate and monitor network infrastructures using Python.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
